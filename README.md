# Kepler-s-Laws-Visualizer
Kepler’s Laws Visualizer is a computational physics project that demonstrates the three laws of planetary motion proposed by Johannes Kepler. The simulator visually represents elliptical orbits, changing orbital speeds, and the relationship between orbital period and distance from the Sun.

# Kepler's Laws Visualizer

## Overview

Kepler's Laws Visualizer is a scientific computing project developed using Python, Pygame, and NumPy. The simulator demonstrates planetary motion and illustrates Kepler's three laws through animated elliptical orbits.

---

## Features

- Elliptical orbit visualization
- Planetary motion animation
- Kepler's Third Law demonstration
- Astronomy simulation
- Scientific visualization

---

## Technologies Used

- Python
- Pygame
- NumPy
